Plugin Desc

The carousel is a list of items, with the ability to scroll inside of a viewing window. This carousel has two parts to the controls. A previous and next button to control regular scrolling. Also, there is an option to use a "control list" which gives the ability to scroll to any position in the carousel without having to manually scroll to that position.

A full description of the javascript implementation and the settings that are avaliable is in the javascript file.
